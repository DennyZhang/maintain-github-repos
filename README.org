一个月前，收到MANNING出版社的邮件(ps: 也是在github上)，关于Review一本物联网书籍的目录，并提出建议。

也因此带来了其他更多的东西，当然不是这里的主题。在这里，我们就不讨论各种骚扰邮件，或者中文合作。从没有想象过，我也可以在英语世界有一片小天地。

这些告诉我们，Github上找一个你擅长的主题，那么会有很多人找上你的。
